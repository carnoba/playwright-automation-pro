**Playwright Automation Pro** is a cutting-edge web automation and scraping engine designed for modern web environments. Built on top of the Microsoft Playwright framework, this system provides a robust solution for end-to-end testing, automated workflows, and high-speed data extraction.

## 🚀 Why Playwright?

Unlike traditional automation frameworks, Playwright offers built-in support for auto-waiting, async execution, and multi-browser contexts. This project leverages these features to deliver a "God-tier" automation experience that is resilient to dynamic web elements and complex JavaScript frameworks.

## ✨ Elite Features

- **Multi-Browser Versatility**: Seamless execution across Chromium, WebKit, and Firefox.
- **Headless & Headed Modes**: Toggle between background processing and visible browser debugging.
- **State Persistence**: Save and reuse browser states (cookies, storage) to bypass repetitive logins.
- **Intelligent Selectors**: Resilient element targeting using text, CSS, and XPath.
- **Automated Workflows**: Pre-configured scripts for lead hunting, form submission, and site auditing.
- **Screenshot & PDF Generation**: Automatic visual evidence and report generation for every task.

## 🛠 Tech Stack

- **Core Engine**: Python
- **Automation Framework**: Playwright
- **Concurrency**: Asyncio for high-performance parallel tasks
- **Reporting**: JSON/CSV Export modules

## ⚙️ Quick Start

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/carnoba/playwright-automation-pro.git
   ```
2. **Install Playwright & Browsers**:
   ```bash
   pip install playwright
   playwright install
   ```
3. **Execute Automation**:
   ```bash
   python "playright bot/main.py"
   ```
